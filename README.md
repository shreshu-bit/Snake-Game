# 🐍 Snake Game

A simple Snake game built using HTML, CSS, and JavaScript.

## 🚀 Live Demo
https://shreshu-bit.github.io/Snake-Game/

## 🎮 Features
- Smooth movement
- Score tracking
- Collision detection
- Classic snake gameplay

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript

## ▶️ How to Run
1. Download the project
2. Open `index.html` in your browser
